How to use FFmpeg

## H.264/AAC Encoding

```
ffmpeg -i input.mov    \
       -c:v libx264    \
       -crf 23         \
       -c:a aac        \
       -b:a 128k       \
       -ar 44100       \
       -map_metadata 0 \
       -preset slow    \
       out.mp4         \
```

### options

* -c:v [ libx264 | libx265 | copy ]
* -crf [ 0 - 51 ]<br>
    0=lossless, 51=worst<br>
    for libx264, default: 23. 18=almost lossless<br>
    for libx265, default: 28.
* -c:a [ libfdk\_aac | aac | copy ]
